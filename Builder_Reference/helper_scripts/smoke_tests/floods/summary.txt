# FLOOD VECTOR ENRICHMENT – DATABASE SUMMARY

## DFO (Dartmouth Flood Observatory)

Provides authoritative flood event records including:

* Start and end dates
* Country and geographic coordinates (lat/long)
* Death and displacement counts
* Severity classification
* GlideNumber (standard disaster identifier)

## EM-DAT (International Disaster Database)

Provides structured disaster records including:

* Standardised start and end dates
* ISO country codes and regional classification
* Total deaths
* Total affected population
* Total economic damage (USD)
* Canonical disaster ID

## GDACS (Global Disaster Alert and Coordination System)

Provides near real-time disaster alert data including:

* Event ID and alert level
* Start timestamp
* Geographic coordinates
* Severity indicators
* Population exposure estimates

## NASA EONET (Earth Observatory Natural Event Tracker)

Provides natural event tracking data including:

* Event ID and category
* Timestamped geographic geometry (lat/lon over time)
* Event status (open/closed)
* Source references

## ReliefWeb (UN OCHA API)

Provides structured humanitarian disaster reporting including:

* Disaster identifiers
* Country information (including ISO codes)
* Publication dates
* Structured disaster classification metadata
* Report titles and text
